# make-site
Creating a Simple File Structure for Websites

```
project name/
└── style/
│   ├── base/
│   │   ├── _mixins.scss
│   │   ├── _reset.scss
│   │   └── _variables.scss
│   └── layouts/
│   │   ├── bootstrap-4-grid.css
│   │   └── bootstrap-3-grid.css
│   └── modules/
│   └── pages/
│   └── style.scss
└── img/
└── js/
└── index.html
└── package.json

```

# Installation

```
npm i -g make-site

```

# Usage

```
make-site -n <project-name>

```
